📧 Bulk Mail Sender Application

A full stack MERN application that allows users to send bulk emails efficiently to multiple recipients. This project is designed to automate email sending and simplify communication workflows.

🚀 Features

- 📩 Send emails to multiple recipients at once
- 📝 Custom email subject and message
- 📄 Add multiple email addresses manually
- ⚡ Fast and efficient email delivery using Nodemailer
- 🔒 Secure backend handling

🛠️ Technologies Used

Frontend:

- React.js
- HTML
- CSS
- JavaScript

Backend:

- Node.js
- Express.js

Database:

- MongoDB (optional)

Other Tools:

- Nodemailer
